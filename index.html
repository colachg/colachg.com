Let’s Encrypt

该网站提供免费的https证书，有效期是三个月，到期可更新。
获取证书方法, 以Ubuntu 16.04为例:

  • 安装certbot：

  • 获取证书：

    sudo certbot certonly --standalone -d example.com -d www.example.com

    域名要提前指向所在主机的公网IP地址.

  • 更新证书(–dry-run只是测试下，正真更新的时候要去掉):

    　sudo certbot renew --dry-run
